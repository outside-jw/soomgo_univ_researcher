\<표 1\> 창의적 문제해결 과정

| 주요 요소 | 세부 단계 | 내용 |
| :---: | :---: | ----- |
| 도전(문제) 이해 | 기회 구성 | 다양한 가능성과 과제를 고려하여 문제 해결을 통해 달성할 수 있는 건설적이고 유익한 목표를 식별하는 단계 |
|  | 자료 탐색 | 여러 관점에서 다양한 자료를 조사하고, 상황의 핵심 요소를 파악하여 문제의 본질을 이해하는 단계 |
|  | 문제 구조화 | 창의적 사고를 자극하는 방식으로 문제를 재구성하여 개방형 질문 형태로 진술하는 단계 |
| 아이디어 생성 | 아이디어 생성 | 유창성, 유연성, 독창성을 바탕으로 다양한 아이디어를 생성하고, 실행 가능성이 높은 아이디어를 선별하는 단계 |
| 실행 준비 | 해결책 고안 | 유망한 아이디어를 전략과 도구를 활용해 개선하여 실행 가능한 해결책으로 구체화하는 단계 |
|  | 수용 구축 | 해결책의 실행을 위해 필요한 지원을 마련하고, 예상되는 어려움을 극복할 방법을 고민하며, 해결책을 점검할 수 있는 구체적인 계획을 세우는 단계 |

(a) 도전 이해하기 단계에서는 문제를 탐색하고 목표를 명확히 하는 과정을 거친다. (b) 아이디어 생성하기 단계에서는 다양한 아이디어를 산출하고, 실행 가능성이 높은 아이디어를 선별한다. (c) 실행 준비하기 단계에서는 유망한 아이디어를 정제하여 실행 가능한 해결책으로 구체화한다.
